# PrimS Marketplace

**PRIMS** - Production Ready Intelligently Managed Synthesis

## Installation

```bash
# Add marketplace (one-time)
/plugin marketplace add prims/marketplace

# Install plugin
/plugin install foundry
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| `foundry` | AI-assisted development workflows with MCP server, commands, agents, and hooks |
